Messaging App

Introducing the "Messaging App," a cutting-edge Flutter application designed for seamless communication in healthcare settings. Tailored for both Admins and Users, this app offers swift user approval, real-time notifications, and advanced features like broadcast messages and scheduled reminders, all within a hospital-themed interface.

Project Overview

For Users, the registration process is swift, with notifications sent to the Admin for approval. Admin holds the power to accept or reject new users, ensuring a secure environment. Once approved, Users receive instant notifications whenever the Admin sends messages. The theme revolves around a hospital setting, incorporating position and department details like Physician, Nurse, RT, Resident, and Radiographer.

Admins wield a comprehensive set of features. They can send broadcast messages to all users, specific users, or designated groups, facilitating efficient communication. Furthermore, Admins can schedule reminder messages, allowing users to view them later in their dedicated reminder section. The app seamlessly integrates Firebase Cloud Messaging (FCM) for both in-app and push notifications, ensuring that important messages are never missed.
